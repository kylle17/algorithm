문제)
   소수란 1보다 크며 1과 자기 자신만을 약수로 가지는 수를 의미합니다. 
   (1보다 커야 하기 때문에 1은 소수가 아니다. )

   어떤 수 X가 소수인지를 판별하려면 
   X를 2부터 X의 제곱근까지의 숫자로 나누어 떨어지는지 검사하면 된다. 
   25는 2, 3, 4, 5로 나누었을 때 5로 나누어 떨어지므로 소수가 아님.
   41는 2, 3, 4, 5, 6으로 나누어도 한 번도 나누어 떨어지지 않으므로 소수이다.

답1) 내가 적은 답. 

#include<stdio.h>

int main(void)
{
	int i=2; int n;
	scanf_s("%d", &n);

	while (1) {
		if (i*i <= n) {
			if (n%i == 0) {
				printf("소수 아님");
				break;
			}
			else {
				i++;
			}
		}
		else {
			printf("소수");
			break;
		}
	}
}

#include<stdio.h>
#include<math.h>

int main(void)
{
	int i; int n;
	scanf_s("%d", &n);

	i = 2;
	while (1) {
		if (i <= sqrt(n)) {
			if (n%i == 0) {
				printf("소수 아님");
				break;
			}
			else {
				i++;
			}
		}
		else {
			printf("소수");
			break;
		}
	}
}
